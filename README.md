# 短视频带货爆款文案工作流 - 项目总览

## 🎯 项目简介

这是一个基于 AI 的爆款文案生成系统，通过深度分析文案的结构、情绪和心理学原理，自动生成多个自然亲切的变体，并提供多语言翻译。系统具备持续学习能力，会积累样本库以提升生成质量。

## 🚀 快速开始

### 1. 直接使用
在 AI 对话框中直接粘贴你的爆款文案，系统会自动：
1. 深度分析文案（结构/情绪/心理学原理）
2. 保存分析洞察到样本库
3. 生成 3-5 个自然亲切的变体
4. 提供多语言翻译（中/英/西/日）

## ✨ 核心特性

### 1. 深度多维度分析
- **结构分析**：开头/中间/结尾的设计逻辑
- **情绪分析**：主导情绪、强度、转折点
- **语言特点**：人称视角、语气风格、修辞手法
- **认知劫持**：稀缺性、社会认同、损失厌恶等心理学原理
- **哲学原理**：底层的人性洞察和价值观

### 2. 智能变体生成
- 保持核心价值，换个角度表达
- 自然亲切，像朋友分享（非营销话术）
- 多样化风格（好奇心驱动/故事叙述/用户见证等）
- 符合 TikTok/抖音短视频场景

### 3. 多语言翻译
- 支持：中文、English、Español、日本語
- 文化适配，非字面翻译
- 保持情绪强度和感染力

### 4. 持续学习能力
- 每次分析自动保存到样本库
- 提取可复用的模式和技巧
- 随着样本增多，生成质量越来越高
- 学习你的偏好和风格

## 📊 工作流程

```
用户粘贴文案
    ↓
深度多维度分析
    ↓
保存到样本库（学习）
    ↓
生成 3-5 个变体
    ↓
多语言翻译
    ↓
保存到 output/
```

## 🔧 技术规范

### 内容质量标准
- **合规性**：避免敏感词、夸大宣传、违规承诺
- **风格要求**：自然亲切，像朋友分享
  - ❌ 避免："限时抢购"、"错过后悔"、"史上最低"
  - ✅ 推荐："我最近发现"、"真的好用"、"分享给你们"

### 翻译质量标准
- 保持营销语气和情绪强度
- 符合目标语言的表达习惯
- 文化适配，非字面翻译

### 样本库规范
- 自动保存每次分析
- 隐私保护（品牌名/产品名用占位符）
- 支持手动管理和查询
