# 短视频带货爆款文案工作流

## 项目简介

基于 AI 的爆款文案生成系统，通过深度分析文案的结构、情绪和心理学原理，自动生成多个自然亲切的变体，并提供多语言翻译和 Fish Audio 配音标签。

## 快速开始

### 基础使用
```
/爆款文案
[粘贴你的爆款文案]
```

系统会自动完成：分析 → 保存样本 → 生成复刻版（推荐）+ 变体 → 输出推荐版四语翻译 → 询问配音/保存

### 常用指令
```
/爆款文案                    # 快速分析（默认）
/爆款文案 --深度             # 深度分析
/爆款文案 --变体数量=5       # 生成5个变体
/爆款文案 --配音             # 含 Fish Audio 配音标签
/配音标签                    # 单独生成配音标签
/样本库                      # 查看样本库
```

### 生成后可用的快捷指令
```
默认已翻译推荐版；如需翻译其他变体：翻译变体2
配音 / 保存
更口语化 / 增强紧迫感 / 换个情绪基调 / 针对年轻人
```

## 核心特性

1. **深度多维度分析** — 结构/情绪/心理学原理/认知劫持
2. **智能变体生成** — 3-5个不同风格的变体
3. **多语言翻译** — 中/英/西/日，文化适配
4. **Fish Audio 配音标签** — 情感语音合成标签
5. **持续学习** — 样本库自动积累，越用越好
6. **反馈优化** — 记录用户偏好，自动调整

## 工作流程

```
用户粘贴文案 → 深度分析 → 保存样本 → 生成变体 → 翻译 → 配音标签（可选）→ 保存到 output/
```
