# 样本保存格式模板

> AI 保存样本到 `samples/cases/` 时使用此格式

---

## 文件命名

`sample-NNN-产品名-文案类型.md`（NNN 为递增编号）

## 样本格式

```markdown
# 产品名 - TikTok文案类型

## 原文

[完整的原始文案内容]

## 关键洞察

1. **洞察1**：[具体描述 + 为什么有效]
2. **洞察2**：[具体描述 + 为什么有效]
3. **洞察3**：[具体描述 + 为什么有效]
4. **不足**：[分析文案的薄弱点，如情绪平淡/缺少CTA/重复用词等]

## 标签

#标签1 #标签2 #标签3

## 元数据

- 收集日期：YYYY-MM-DD
- 来源平台：TikTok/抖音/用户提供
- 产品类型：类别
- 情绪强度：N/10
```

## index.json 对应条目

保存样本文件后，同步在 `samples/index.json` 的 `samples` 数组中添加：

```json
{
  "id": "sample-NNN",
  "category": "产品类别",
  "product": "产品名",
  "date": "YYYY-MM-DD",
  "source": "来源",
  "tags": ["标签1", "标签2"],
  "emotion_score": 7,
  "key_patterns": ["模式1", "模式2"],
  "summary": "一句话摘要"
}
```

并将 `total_samples` 加 1。

## 隐私保护

如果文案中包含具体品牌名或产品名，在样本中用 `[品牌名]`、`[产品名]` 等占位符替换。
