# 爆款文案生成器使用指南

## 快速开始

### 1. 最简单的使用方式

直接在对话框中粘贴你的爆款文案，AI 会自动识别并开始分析：

```
[直接粘贴文案内容]
```

AI 会自动：
1. 深度分析文案的结构、情绪、心理学原理
2. 保存分析洞察到样本库
3. 生成 3-5 个自然亲切的变体
4. 提供多语言翻译

### 2. 使用中文指令触发

如果 AI 没有自动识别，可以使用以下指令：

#### 主指令
- `/爆款文案` - 完整流程（推荐）
- `/文案生成` - 完整流程
- `/vc` - 简写

#### 快捷模式
- `/爆款文案 --快速` - 跳过详细分析，快速生成
- `/爆款文案 --仅分析` - 只分析不生成变体
- `/爆款文案 --仅翻译` - 只翻译已有文案
- `/爆款文案 --批量` - 批量处理多段文案

#### 参数化使用
- `/爆款文案 --变体数量=5` - 生成5个变体（默认3-5个）
- `/爆款文案 --风格=专业` - 指定风格（友好/专业/轻松）
- `/爆款文案 --语言=zh,en,es` - 指定翻译语言
- `/爆款文案 --跳过保存` - 不保存到样本库

#### 组合使用示例
```
/爆款文案 --变体数量=5 --风格=专业
/文案生成 --仅分析
/vc --快速 --跳过保存
```

#### 自然语言触发
也可以直接说：
- "帮我分析这段文案"
- "生成文案变体"
- "这是一段爆款文案，帮我改写"

## 使用场景

### 场景 1：分析爆款文案
**你的操作**：粘贴一段在 TikTok/抖音上表现很好的文案

**AI 会做什么**：
- 深度解构文案的成功要素
- 提取认知劫持和心理学原理
- 保存关键洞察供未来参考

### 场景 2：生成变体文案
**你的操作**：提供原始文案

**AI 会做什么**：
- 基于分析生成 3-5 个不同风格的变体
- 每个变体都保持核心价值，但换个角度表达
- 风格自然亲切，像朋友分享

### 场景 3：多语言翻译
**你的操作**：在生成变体后，回复"翻译"

**AI 会做什么**：
- 提供中文、英文、西班牙语、日语版本
- 不是字面翻译，而是文化适配
- 保持原文的情绪强度和感染力

### 场景 4：持续学习
**你的操作**：不断提供新的爆款文案

**AI 会做什么**：
- 每次分析都会保存到样本库
- 随着样本增多，生成质量越来越高
- 学习你的偏好和风格

## 高级用法

### 对比分析
提供 2 段文案，AI 会自动对比分析：
```
文案A：[内容]

文案B：[内容]

帮我对比分析这两段文案
```

### 定向优化
告诉 AI 你想要的调整方向：
- "更口语化一点"
- "增强紧迫感"
- "换个情绪基调"
- "针对年轻人群体"

### 反馈学习
告诉 AI 哪些变体效果好：
```
变体2的转化率很高，以后多生成这种风格
```

## 输出文件说明

### 样本库（samples/）
- `samples/index.json`：所有样本的索引
- `samples/cases/`：每个分析过的文案详细记录
- `samples/template.md`：样本格式模板

### 输出目录（output/）
生成的文案会保存到这里，文件命名格式：
```
20260304-143022-好奇心驱动.md
```

## 注意事项

### 内容合规
- 避免敏感词、夸大宣传
- 不编造数据和效果
- 符合平台内容政策

### 风格要求
- ❌ 避免：营销话术（"限时抢购"、"错过后悔"）
- ✅ 推荐：朋友分享（"我最近发现"、"真的好用"）

### 隐私保护
如果文案中包含具体品牌名或产品名，在样本库中会用占位符替换，保护隐私。

## 常见问题（FAQ）

### 基础使用

**Q: AI 没有自动识别我的文案怎么办？**
A: 使用明确的指令触发，如 `/爆款文案` 或 `/文案生成`，然后粘贴你的文案。

**Q: 生成的变体不满意怎么办？**
A: 告诉 AI 具体哪里不满意，比如：
- "太正式了" → AI 会生成更口语化的版本
- "不够有趣" → AI 会增加趣味性和吸引力
- "换个情绪基调" → AI 会调整主导情绪
- "针对年轻人群体" → AI 会调整为年轻人喜欢的风格

**Q: 可以批量处理多段文案吗？**
A: 可以！使用 `/爆款文案 --批量` 指令，然后一次性提供多段文案。AI 会询问你想逐个处理还是批量处理。

**Q: 如何只分析不生成变体？**
A: 使用 `/爆款文案 --仅分析` 指令，AI 会只进行深度分析，不生成变体。

**Q: 如何指定生成的变体数量？**
A: 使用参数 `--变体数量=N`，例如：`/爆款文案 --变体数量=5` 会生成5个变体。

### 样本库管理

**Q: 样本库会越来越大吗？**
A: 是的，但这是好事！样本越多，AI 生成的质量越高。系统会自动学习历史样本中的成功模式。

**Q: 如何查看历史样本？**
A: 有三种方式：
1. 查看 `samples/index.json` 文件（JSON格式索引）
2. 直接浏览 `samples/cases/` 目录（每个样本一个文件）
3. 使用指令 `/样本库` 查看统计和列表（即将推出）

**Q: 可以不保存到样本库吗？**
A: 可以，使用 `--跳过保存` 参数，如：`/爆款文案 --跳过保存`

**Q: 样本库中的品牌名会被保护吗？**
A: 是的，系统会自动将品牌名、产品名替换为占位符（如 `[品牌名]`），保护隐私。

### 翻译功能

**Q: 支持哪些语言的翻译？**
A: 默认支持：中文（简体）、英文、西班牙语、日语。可以通过 `--语言` 参数指定，如：`--语言=zh,en,ja`

**Q: 翻译是字面翻译吗？**
A: 不是！AI 会进行文化适配，确保翻译后的文案符合目标语言的表达习惯，保持原文的情绪强度和感染力。

**Q: 可以只翻译某个变体吗？**
A: 可以，在生成变体后，回复 "翻译变体2" 即可只翻译第2个变体。

### 高级功能

**Q: 如何对比分析两段文案？**
A: 提供两段文案，并说明 "帮我对比分析这两段文案"，AI 会自动进入对比模式。

**Q: AI 会记住我的偏好吗？**
A: 会的！系统会学习你的反馈，记住你喜欢的风格和避免的词汇，越用越符合你的需求。

**Q: 如何恢复上次未完成的工作？**
A: 使用 `/恢复会话` 指令，AI 会自动加载上次的工作状态。

**Q: 生成的文案保存在哪里？**
A: 保存在 `output/` 目录，文件命名格式为：`YYYYMMDD-HHMMSS-变体标签.md`

### 技术问题

**Q: 为什么有时候文件操作失败？**
A: 如果路径中包含中文，确保系统使用的是 Git Bash（不是 PowerShell）。项目已配置强制使用 Git Bash。

**Q: 如何修改默认配置？**
A: 编辑 `.ai/config.yaml` 文件，可以修改默认变体数量、风格偏好、避免的词汇等。

**Q: 支持哪些 AI 工具？**
A: 兼容 Claude Code、Kiro、Cursor 等支持 Skills 的 AI 编程工具。

### 内容质量

**Q: 如何确保生成的文案合规？**
A: 系统内置合规检查，会自动避免敏感词、夸大宣传。你也可以在 `.ai/config.yaml` 中设置 `compliance_level`（strict/moderate/relaxed）。

**Q: 生成的文案会重复吗？**
A: 不会。系统会确保每个变体都有独特的角度和表达方式，避免重复。

**Q: 如何让文案更自然？**
A: 使用 `--风格=友好` 参数，或者在生成后告诉 AI "更口语化一点"、"像朋友聊天一样"。

## 最佳实践

1. **提供完整文案**：至少 20 字以上，内容完整
2. **告诉 AI 效果**：如果某个文案表现好，告诉 AI 数据
3. **持续积累**：多提供爆款文案，让系统学习
4. **及时反馈**：生成的变体不满意就说出来，AI 会调整
