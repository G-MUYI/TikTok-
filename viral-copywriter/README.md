# 爆款文案生成器使用指南

## 快速开始

### 1. 最简单的使用方式

直接在对话框中粘贴你的爆款文案，AI 会自动识别并开始分析：

```
[直接粘贴文案内容]
```

AI 会自动：
1. 深度分析文案的结构、情绪、心理学原理
2. 保存分析洞察到样本库
3. 生成 3-5 个自然亲切的变体
4. 提供多语言翻译

### 2. 明确触发方式

如果 AI 没有自动识别，可以明确说：
- "帮我分析这段文案"
- "生成文案变体"
- "这是一段爆款文案，帮我改写"

## 使用场景

### 场景 1：分析爆款文案
**你的操作**：粘贴一段在 TikTok/抖音上表现很好的文案

**AI 会做什么**：
- 深度解构文案的成功要素
- 提取认知劫持和心理学原理
- 保存关键洞察供未来参考

### 场景 2：生成变体文案
**你的操作**：提供原始文案

**AI 会做什么**：
- 基于分析生成 3-5 个不同风格的变体
- 每个变体都保持核心价值，但换个角度表达
- 风格自然亲切，像朋友分享

### 场景 3：多语言翻译
**你的操作**：在生成变体后，回复"翻译"

**AI 会做什么**：
- 提供中文、英文、西班牙语、日语版本
- 不是字面翻译，而是文化适配
- 保持原文的情绪强度和感染力

### 场景 4：持续学习
**你的操作**：不断提供新的爆款文案

**AI 会做什么**：
- 每次分析都会保存到样本库
- 随着样本增多，生成质量越来越高
- 学习你的偏好和风格

## 高级用法

### 对比分析
提供 2 段文案，AI 会自动对比分析：
```
文案A：[内容]

文案B：[内容]

帮我对比分析这两段文案
```

### 定向优化
告诉 AI 你想要的调整方向：
- "更口语化一点"
- "增强紧迫感"
- "换个情绪基调"
- "针对年轻人群体"

### 反馈学习
告诉 AI 哪些变体效果好：
```
变体2的转化率很高，以后多生成这种风格
```

## 输出文件说明

### 样本库（samples/）
- `samples/index.json`：所有样本的索引
- `samples/cases/`：每个分析过的文案详细记录
- `samples/template.md`：样本格式模板

### 输出目录（output/）
生成的文案会保存到这里，文件命名格式：
```
20260304-143022-好奇心驱动.md
```

## 注意事项

### 内容合规
- 避免敏感词、夸大宣传
- 不编造数据和效果
- 符合平台内容政策

### 风格要求
- ❌ 避免：营销话术（"限时抢购"、"错过后悔"）
- ✅ 推荐：朋友分享（"我最近发现"、"真的好用"）

### 隐私保护
如果文案中包含具体品牌名或产品名，在样本库中会用占位符替换，保护隐私。

## 常见问题

**Q: AI 没有自动识别我的文案怎么办？**
A: 明确说"帮我分析这段文案"或"生成文案变体"

**Q: 生成的变体不满意怎么办？**
A: 告诉 AI 具体哪里不满意，比如"太正式了"、"不够有趣"

**Q: 可以批量处理多段文案吗？**
A: 可以，一次性提供多段，AI 会询问你想逐个处理还是批量处理

**Q: 样本库会越来越大吗？**
A: 是的，但这是好事！样本越多，AI 生成的质量越高

**Q: 如何查看历史样本？**
A: 查看 `samples/index.json` 或直接浏览 `samples/cases/` 目录

## 最佳实践

1. **提供完整文案**：至少 20 字以上，内容完整
2. **告诉 AI 效果**：如果某个文案表现好，告诉 AI 数据
3. **持续积累**：多提供爆款文案，让系统学习
4. **及时反馈**：生成的变体不满意就说出来，AI 会调整
