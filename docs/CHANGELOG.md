# 更新日志 (CHANGELOG)

## [1.3.0] - 2026-03-04

### 新增功能 ✨

#### 1. 执行模板系统 📋
- **analysis-output.md** - 文案分析阶段的标准输出模板
- **variant-output.md** - 变体生成阶段的标准输出模板
- **translation-output.md** - 多语言翻译的标准输出模板
- **sample-save.md** - 样本保存的格式模板

**作用**：
- 确保AI输出格式的一致性
- 提供清晰的结构参考
- 便于用户理解和使用

#### 2. 反馈收集机制 💬
- **主动询问**：生成变体后主动询问用户偏好
- **记录反馈**：将用户反馈保存到 `.ai/user-profile.json`
- **应用学习**：下次生成时优先使用用户喜欢的模式
- **避免重复**：记住用户不喜欢的词汇和表达方式

**反馈处理流程**：
- 用户说"我最喜欢变体2" → 记录偏好的标签和特点
- 用户说"太正式了" → 记录避免的风格
- 用户说"更口语化" → 立即重新生成，并记录偏好
- 用户说"变体3转化率很高" → 记录高效模式，提高权重

#### 3. 样本库管理功能 📚
新增样本管理指令：
- `/样本库` - 查看样本库统计信息
- `/样本库 列表` - 列出所有样本
- `/样本库 搜索 [关键词]` - 搜索样本
- `/样本库 分类 [类别]` - 按类别查看样本（美妆/食品/数码等）
- `/样本库 导出` - 导出样本库为报告

**功能详情**：
- 统计信息：总样本数、分类统计、最受欢迎的模式
- 样本列表：按时间倒序显示，包含摘要和标签
- 搜索功能：在原始文案、分析结果、标签中搜索
- 分类查看：筛选特定类别的样本
- 导出报告：生成样本库汇总分析报告

### 使用示例 💡

#### 使用执行模板
AI在生成输出时会参考模板，确保格式一致：
```
📊 **文案深度分析**
[按照 analysis-output.md 模板输出]

🎯 **生成的文案变体**
[按照 variant-output.md 模板输出]
```

#### 反馈收集
```
你: /爆款文案
[AI生成3个变体]

AI: 💬 请告诉我：
1. 你最喜欢哪个变体？
2. 需要调整的地方？

你: 我最喜欢变体2，更口语化，以后多生成这种风格

AI: ✅ 已记录你的偏好！下次生成时会优先使用口语化风格。
```

#### 样本库管理
```
/样本库
📊 总样本数：15
   美妆类：5 | 食品类：6 | 数码类：4

/样本库 搜索 面膜
📚 找到2个相关样本...

/样本库 导出
✅ 已导出样本库报告到 output/sample-library-report-20260304.md
```

## [1.1.0] - 2026-03-04

### 新增功能 ✨

#### 1. 中文指令系统
- **主指令**：`/爆款文案`、`/文案生成`（中文优先）
- **快捷模式**：
  - `/爆款文案 --快速` - 快速生成模式
  - `/爆款文案 --仅分析` - 只分析不生成
  - `/爆款文案 --仅翻译` - 只翻译
  - `/爆款文案 --批量` - 批量处理
- **参数支持**：
  - `--变体数量=N` - 指定生成数量
  - `--风格=友好|专业|轻松` - 指定风格
  - `--语言=zh,en,es,ja` - 指定翻译语言
  - `--跳过保存` - 不保存到样本库

#### 2. 全局配置系统
