# 快速参考卡片

> 爆款文案生成器 - 常用指令速查

---

## 🚀 基础指令

```bash
/爆款文案              # 启动文案生成（默认快速模式）
/文案生成              # 同上
/vc                   # 简写
```

---

## ⚙️ 分析挡位

| 挡位 | 指令 | 时长 | 适用场景 |
|------|------|------|---------|
| 🚀 快速 | `--快速` 或默认 | 10秒 | 日常快速生成 |
| ⚡ 标准 | `--标准` | 15秒 | 重要项目 |
| 🔬 深度 | `--深度` | 20-30秒 | 学习研究 |

**示例**：
```bash
/爆款文案                # 快速模式（默认）
/爆款文案 --标准         # 标准模式
/爆款文案 --深度         # 深度模式
```

---

## 🎛 常用参数

| 参数 | 说明 | 示例 |
|------|------|------|
| `--变体数量=N` | 生成N个变体 | `--变体数量=5` |
| `--风格=X` | 指定风格 | `--风格=专业` |
| `--语言=X,Y` | 指定翻译语言 | `--语言=ja,en` |
| `--跳过保存` | 不保存到样本库 | `--跳过保存` |
| `--仅分析` | 只分析不生成 | `--仅分析` |
| `--仅翻译` | 只翻译已有文案 | `--仅翻译` |

---

## 📝 组合示例

```bash
# 标准分析 + 5个变体 + 专业风格
/爆款文案 --标准 --变体数量=5 --风格=专业

# 深度分析 + 只分析不生成
/爆款文案 --深度 --仅分析

# 快速模式 + 不保存样本
/爆款文案 --快速 --跳过保存

# 只翻译成日语和英语
/爆款文案 --仅翻译 --语言=ja,en
```
