# Skills 索引

## 可用 Skills

### 1. 爆款文案生成器 (viral-copywriter)

深度分析爆款文案，生成多个自然亲切的变体，并提供多语言翻译和配音标签。

- **主指令**：`/爆款文案`、`/文案生成`、`/vc`
- **配音标签**：`/配音标签`、`/fish-audio`
- **样本库**：`/样本库`
- **详细文档**：[viral-copywriter/SKILL.md](../viral-copywriter/SKILL.md)

### 2. 项目初始化工具 (dev-workflow-init)

通过对话式访谈收集需求，生成 AGENTS.md 规则文件。

- **指令**：`/初始化`、`/启动项目`、`/init`
- **详细文档**：[skills/dev-workflow-init/README.md](skills/dev-workflow-init/README.md)

## 全局命令

| 指令 | 说明 |
|------|------|
| `/帮助` / `/help` | 显示帮助信息 |

## 配置

全局配置文件：`.ai/config.yaml`

可自定义：默认变体数量、风格偏好、避免词汇、翻译语言等。
