## 📚 可用 Skills

### 1. 爆款文案生成器 (viral-copywriter)

**功能**：深度分析爆款文案，生成多个自然亲切的变体，并提供多语言翻译

**中文指令**：
- `/爆款文案` - 完整流程（推荐）
- `/文案生成` - 完整流程
- `/vc` - 简写

**快捷模式**：
- `/爆款文案 --快速` - 跳过详细分析
- `/爆款文案 --仅分析` - 只分析不生成
- `/爆款文案 --仅翻译` - 只翻译
- `/爆款文案 --批量` - 批量处理

**样本管理指令** ⭐ 新增：
- `/样本库` - 查看样本库统计
- `/样本库 列表` - 列出所有样本
- `/样本库 搜索 [关键词]` - 搜索样本
- `/样本库 分类 [类别]` - 按类别查看
- `/样本库 导出` - 导出样本库报告

**参数**：
- `--变体数量=N` - 生成N个变体
- `--风格=友好|专业|轻松` - 指定风格
- `--语言=zh,en,es,ja` - 指定翻译语言
- `--跳过保存` - 不保存到样本库

**使用示例**：
```
/爆款文案
[粘贴你的文案]

/爆款文案 --变体数量=5 --风格=专业
[粘贴你的文案]

/文案生成 --仅分析
[粘贴你的文案]
```

**详细文档**：[查看完整说明](../viral-copywriter/README.md)

## 🔄 会话持久化 ⭐ 新功能

### 功能说明

AI会自动记住你的工作状态和偏好，即使关闭窗口后重新打开，也能继续之前的工作。

### 自动恢复

AI启动时会自动检查状态文件，如果发现未完成的工作会主动询问：

```
AI: 👋 欢迎回来！

我注意到上次你分析了一段美妆类文案，已生成3个变体但还未翻译。

你想：
1. 继续翻译上次的变体
2. 重新开始新的分析
3. 查看上次的分析结果
```

### 会话管理指令

```bash
/恢复会话    # 恢复上次未完成的工作
/查看历史    # 查看最近的工作记录
/清空状态    # 清空会话状态，重新开始
/统计        # 查看使用统计
```

### 用户偏好学习

AI会学习你的偏好并自动应用：
- 记住你喜欢的文案风格
- 避免你不喜欢的词汇
- 根据历史反馈优化生成策略

### 状态文件

- `.ai/session-state.json` - 当前会话状态（AI自动维护）
- `.ai/user-profile.json` - 用户偏好配置（可手动编辑）
- `.ai/work-history.json` - 工作历史记录（AI自动维护）

详细说明请查看：[SESSION_PERSISTENCE.md](../docs/SESSION_PERSISTENCE.md)

## 🚨 重要提示

### Windows 中文路径处理

本项目**强制使用 Git Bash**，禁用 PowerShell，以确保中文路径正确处理。

所有文件操作都会：
- 使用正斜杠 `/`
- 路径用双引号包裹
- 支持中文文件名和目录名

示例：
```bash
# 创建中文目录
mkdir -p "samples/案例库/美妆类"

# 创建中文文件
touch "output/生成文案_美妆类_20260304.md"
```

### Skills 触发优先级

1. **显式指令** > 自动识别
   - 使用 `/爆款文案` 等指令会立即触发
   - 直接粘贴文案会根据内容判断

2. **上下文感知**
   - 如果正在使用某个 Skill，会优先继续该 Skill
   - 避免意外切换
